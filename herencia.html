Inheritance Concepts in Object Oriented Programming

Superclass and Subclass

The superclass It is the base class from which other classes are derived. The subclass inherits attributes and methods from the superclass and can extend or modify its behavior.

Example: If we have a class Animal as a superclass, a subclass Dog can inherit from Animal and add specific methods such as bark(). 

class Animal {
                        constructor(name) {
                            this.name = name;
                            }
                            makeSound() {
                            console.log('Generic Sound');
                            }
                            }
                            
                            class Dog extends Animal {
                            bark() {
                            console.log('Woof');
                            }
}

Code Reuse

Code reuse is one of the main advantages of inheritance. It allows subclasses to use methods and attributes of the superclass, avoiding duplication and making code maintenance easier.

Example: If Animal has a makeSound method, any subclass such as Cat or Dog can use this method directly. 

class Animal {
                        makeSound() {
                        console.log('Animal Sound');
                        }
                        }
                        
                        class Cat extends Animal {
                        // Reuses the makeSound method
                        }
                        
                        const myCat = new Cat();
                        myCat.makeSound(); // Prints: Animal Sound

Method Overrides

Method overriding allows a subclass to provide a specific implementation of a method that is already defined in the superclass.

Example: If Animal has a method makeSound, the subclass Dog can override this method to make the dog make a bark instead of a generic sound. 

class Animal {
makeSound() {
console.log('Generic sound');
    }
}

class Dog extends Animal {
    makeSound() {
    console.log('Woof');
    }
    }
    
    const myDog = new Dog();
    myDog.makeSound(); // Prints: Woof
