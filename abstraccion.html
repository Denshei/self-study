Abstraction

Abstraction is like a way to simplify complex things by focusing only on what is most important. Imagine you are building a toy and only care about its design and how it works, without worrying about every small piece it consists of. That's what abstraction does in programming: it helps us focus on the essentials and set aside details that we don't need at the moment.

In programming, we use abstraction to create components that perform specific tasks without needing to know how they work internally. For example, when we use a calculator in an application, we only see the buttons and the screen, but we donâ€™t need to know how the internal calculations are done. This makes it easier to work with complex systems.

We also use abstraction in our daily lives. For example, when we think of "fruit," we don't care if it's an apple or an orange, we only care that it is something we eat. This way, we can treat different fruits similarly without thinking about each specific type.
